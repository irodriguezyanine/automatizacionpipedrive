# Tabla de agentes – Awesome Claude Code Subagents

## ¿Hay agente para LinkedIn?

**No.** En este repositorio **no existe un agente llamado "LinkedIn"** ni uno dedicado solo a LinkedIn.

**Los que más se acercan a lo que podrías hacer con LinkedIn:**

| Si quieres… | Usa este agente |
|-------------|------------------|
| Escribir posts, artículos y contenido para redes (incluido LinkedIn) | **content-marketer** |
| Material de ventas, propuestas y mensajes para prospección | **sales-engineer** |
| Mejorar visibilidad y palabras clave (perfil, contenido) | **seo-specialist** |
| Definir estrategia o procesos (ej. automatización LinkedIn) | **business-analyst** |

Para un “agente que use LinkedIn” (publicar, conectar, analizar) tendrías que **crear tu propio agente** (template del repo o Cursor/Claude) o usar una herramienta/integración específica de LinkedIn.

## Cómo instalar

- **Por categoría (plugin):**  
  `claude plugin marketplace add VoltAgent/awesome-claude-code-subagents`  
  Luego: `claude plugin install voltagent-core-dev` (o `voltagent-lang`, `voltagent-biz`, etc.).

- **Un solo agente (manual):**  
  Copiar el `.md` del agente a `~/.claude/agents/` (global) o `.claude/agents/` (proyecto).

- **Instalador interactivo:**  
  Clonar el repo y ejecutar `./install-agents.sh`.
